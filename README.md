This web app helps you **transcribe meetings in any language into English** and automatically generates **summaries, key discussion points, decisions, and next steps**. It works with both audio and video files.

---

## Features
- Transcribe **audio or video** from 100+ languages
- Translate speech into **English**
- Generate **meeting summaries** with key points, decisions, and action items
- Export **bilingual meeting minutes** in Markdown
- Easy-to-use **web interface** with Streamlit
- Fast processing using Whisper models

---

## Installation

1. Clone this repo:
```bash
git clone https://github.com/Dhriti2220/Meeting-minutes-summarizer.git
cd Meeting-minutes-summarizer

2. Install dependencies:
pip install -r requirements.txt

3. Install ffmpeg for video processing:
# Ubuntu/Debian
sudo apt-get install ffmpeg
# macOS
brew install ffmpeg

How to Use
Run the app:
streamlit run app.py

1. Upload your audio or video file
2. Select a model size (tiny, base, small, medium)
3. Click "Transcribe, Translate & Summarize"
4. View or download the English transcript and summary
